# AI-Based Hiring Prediction System

## Project Objective
Predict whether a candidate will be Hired or Rejected based on resume information.

## Dataset Features
- Skills
- Experience Years
- Education
- Certifications
- Job Role
- Salary Expectation
- Projects Count

## Machine Learning Workflow
1. Data Generation
2. Data Analysis
3. Data Preprocessing
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Prediction

## Algorithm Used
Random Forest Classifier

## Libraries
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
